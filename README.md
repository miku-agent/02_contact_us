## 🚀 Getting Started

To start the development server, run:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

---

## ⚙️ Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS + DaisyUI
- **Database:** Prisma ORM
- **Testing:** Vitest + Playwright
